## Online Judge Server

Heroku URL: https://bits-judge.herokuapp.com/

## Setup

- Install Node.JS from https://github.com/nvm-sh/nvm
- Open this project folder in a terminal
- Run `npm install`
- Run `npm i -g eslint`

## Deployment

- Install the Heroku CLI.
- Commit any changes.
- Run `git push heroku`.
